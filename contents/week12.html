Pre-Tutorial Learning Activities

Relates XP/Scrum to your project [Individual, 3pts]

Pick task (a) or (b). Ensure at least one member picks each task and no more than one member picks the same concept/rule.

Preferably, try to pick concepts/rules that you think are not applicable to the project.

  1. Relate* three key concepts of Scrum to your project.
    e.g. product backlog, product owner, sprints, sprint backlog, daily scrum, user stories, scrumboard, story points, burndown charts, team velocity, scrum master, etc.
  2. Relate three XP rules to your project.

There is no need to bring any written answers; it is good enough if you can explain your answer verbally.

*“Relate” here means: Is it applicable in your project? If not, why not? Are you using something similar or an alternative in your project? etc.

Relate reuse mechanisms to your project [Team, 2pts]

Find a platform, a framework, and a library you could have used in the project. One example each for the whole team is enough.

However, all members should know those examples and why each one is considered a platform, a framework, or a library.

Contribute to V0.4 [Individual, 2pts]

Commit code for this version.
Try to ensure your latest version of the product works before commiting code. i.e. do not commit code that breaks the product.
You are encouraged to add tags in the repo to identify intermediate working versions. e.g. V0.3.1. That way, you can fall back on the latest working version if the intended V0.4 was not reached by the deadline.

Reach V0.4, with time to spare [Team, 3+2=5pts]

[3pts] V0.4 should be noticeably better than the previous version, The code should be tagged as V0.4 in the repo.

[2pts] Create an executable jar file or the exe file (i.e. should be able to run outside Eclipse/Visual Studio).

[+1 bonus point] Reach V0.4 with at least 12 hours in advance.

Dogfood your app [Individual, 2pts]

Sincerely dogfood your app from now on.

Collate code

Mark your code and use the collate tool to collate your code. See V0.4 [release candidate] Additional Requirements To find more about the Collate tool.

Warning: If the collate tool cannot extract your code from the final submission, you are going to lose 10 marks from your final grade.
